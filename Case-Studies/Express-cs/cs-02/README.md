# Greenfield Community Center - Express Project

This project demonstrates how to organize an Express application using a modular structure.

How to Run

1. Install dependencies

npm install

2. Start the server

node app.js

Server runs at

http://localhost:3000


How to Test

Open browser and visit:

Home Page
http://localhost:3000

Events
http://localhost:3000/events

Contact
http://localhost:3000/contact
