# DesignHub - Zustand Slices

Implements the Notification Slice challenge from "Zustand Slices and Modular State Architecture" case study. Features modular state slices for users, files, and notifications with type-safe composition.

## Run

`npm install` then `npm run dev`

## Output

Login, create/delete files, view notifications, mark as read, clear all. State persists on refresh.
