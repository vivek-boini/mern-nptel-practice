# CollabNotes - Advanced Zustand

Implements "Advanced State Management with Zustand" case study with middleware (persist, devtools, immer, logging), state versioning/migration, and React Query integration for async data fetching.

## Run

`npm install` then `npm run dev`

## Output

Change preferences, login, add/edit/delete notes, view collaborators. All persisted state survives refresh. Check console for state logs and Redux DevTools for full state inspection.
